# Multi-Swarm Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Support 2–3 independent 4-agent swarms from a single repo, each with its own iTerm2 sessions and active project.

**Architecture:** Each `launch.sh` invocation auto-assigns a swarm number, creates `swarms/N/` with `pane-config.sh` and `ACTIVE_PROJECT`, and injects `SWARM_ID=N` into each pane's environment. `send-to-agent.sh` reads `$SWARM_ID` to source the correct session config. Agent instructions are updated to reference the per-swarm `ACTIVE_PROJECT` path.

**Tech Stack:** Bash, AppleScript (iTerm2), macOS

### Task 4: Update `launch.sh` — project setup prompt

**Files:**
- Modify: `launch.sh`

- [ ] **Step 1: Add the project setup prompt section**

Find the summary output block — the `echo ""` that precedes `echo "✓ iTerm2 workspace launched"` (around line 158). Insert the following block **before** the `SESSION_IDS=$(osascript ...` block — specifically, right after the skip-agents prompt block ends (after the `SKIP_AGENTS=...` line, around line 80):

```bash
# ── Project Setup ──────────────────────────────────────────────────────────────
echo "── Project Setup ─────────────────────────────────────────────────────────"
echo "  Swarm $SWARM_ID"
echo ""
read -p "  New project or resume existing? [n/r]: " RAW_PROJECT_MODE
echo "──────────────────────────────────────────────────────────────────────────"
echo ""
PROJECT_MODE=$(echo "${RAW_PROJECT_MODE:-n}" | tr '[:upper:]' '[:lower:]' | tr -d '[:space:]')

ACTIVE_PROJECT_VALUE=""
if [ "$PROJECT_MODE" = "r" ]; then
  # List projects/ subdirs excluding inbox
  PROJECT_LIST=()
  while IFS= read -r d; do
    name=$(basename "$d")
    [ "$name" = "inbox" ] && continue
    PROJECT_LIST+=("$name")
  done < <(find "$SCRIPT_DIR/projects" -mindepth 1 -maxdepth 1 -type d | sort)

  if [ ${#PROJECT_LIST[@]} -eq 0 ]; then
    echo "  No existing projects found. Starting as new."
  else
    echo "  Existing projects:"
    for i in "${!PROJECT_LIST[@]}"; do
      echo "    $((i+1))) ${PROJECT_LIST[$i]}"
    done
    echo ""
    read -p "  Pick a project number: " RAW_PROJECT_NUM
    PROJECT_NUM=$(echo "${RAW_PROJECT_NUM}" | tr -d '[:space:]')
    if [[ "$PROJECT_NUM" =~ ^[0-9]+$ ]] && [ "$PROJECT_NUM" -ge 1 ] && [ "$PROJECT_NUM" -le "${#PROJECT_LIST[@]}" ]; then
      ACTIVE_PROJECT_VALUE="${PROJECT_LIST[$((PROJECT_NUM-1))]}"
      echo "  Resuming: $ACTIVE_PROJECT_VALUE"
    else
      echo "  Invalid selection. Starting as new."
    fi
  fi
  echo ""
fi
```
